# Bluestock Mutual Fund Analytics Capstone

## Project Overview

This project aims to build a complete Mutual Fund Analytics platform covering:

* Data Ingestion
* ETL Pipeline
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Performance Analytics
* SQLite Data Warehouse
* Interactive Dashboard
* Advanced Risk Analytics

The project is being developed as part of the Bluestock Data Analyst Internship Program.
